> ### Find the highest value in an array!


## Install

  

```sh
npm i highest
```

##  Usage

Here is a basic example of how `highest` works:

```js
const opposite = require("highest")

console.log(highest(2,3,4)) //4
console.log(highest([2,54,2])) //54
```
